Home / Doc / Docs - Accordion

To use accordian, connect accordian.css and js/accordian.js

  • .accordion - main wrapper
  • .acc-btn - item title
  • .acc-content - content wrapper
  • .acc-content-inner - content inner wrapper
  • .acc-content .open - first child is open
  • .selected - selected title
