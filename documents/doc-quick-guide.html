Home / Doc / Quick Start Guide

Please spend some time reviewing our documents. We believe it will help you to save time in the future.

Our theme is based on bootstrap framework and uses all its components. Please DO NOT update / edit Bootstrap's CSS files; it will interrupt our custom css files and make the website function erratically.

Our theme has modular structure, so web developers don't have to connect a couple of big files that contain dozens of unused elements. But this brings some complexity. You have to connect specific scripts/styles to use their functionality. We have provided quite a few examples of such scripts / styles insertion for your understanding. Also we populate our docs with new pages regularly; these docs contain information about scripts you need to connect as well.
