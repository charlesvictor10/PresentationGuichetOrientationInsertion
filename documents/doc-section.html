Home / Doc / Docs - Section

To create section style, you need to use theme.css

To set the width of section wrapper to 100%, you need to use .sec-row class.

We have provided five section styles for section wrapper. You can change the color and parallax image of section wrapper background.

Section Wrapper Styles

  • .sec-gray-light - gray section color scheme
  • .sec-gray-dark - dark gray section color scheme
  • .sec-black - black section color scheme
  • .sec-blue - blue section color scheme
  • .sec-bg - parallax section contains an image with fixed position in section wrapper background.
