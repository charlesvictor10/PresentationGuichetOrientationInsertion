Titlebars and Breadcrumbs

Home / Doc / Titlebars and Breadcrumbs

To create section style, you need to use theme.css

To set the width of section wrapper to 100%, you need to use below mentioned classes as per the details provided.

For Bottom Breadcrumb

  • .breadcrumb-1
  • .breadcrumb-2
  • .breadcrumb-3

For Right Breadcrumb

  • .breadcrumb-4
  • .breadcrumb-5
  • .breadcrumb-6

For Top Breadcrumb

  • .breadcrumb-7
  • .breadcrumb-8
  • .breadcrumb-9
